faqSections:
  - title: "General FAQs"
    items:
      - question: "What is SkayaJS?"
        tags: ["General"]
        answer: |
          SkayaJS is a React Component Library that wraps browser Navigator APIs like:
          <ul>
            <li>Geolocation</li>
            <li>Bluetooth</li>
            <li>Clipboard</li>
            <li>Media Devices</li>
          </ul>
          into beautiful, themed, and developer-friendly React components and hooks.
      - question: "How do I install SkayaJS?"
        tags: ["General", "Installation"]
        answer: |
          Install via npm or yarn:
          <pre><code>npm install skayajs</code></pre>
          <pre><code>yarn add skayajs</code></pre>
      - question: "Can I use SkayaJS with MUI or Tailwind?"
        tags: ["General", "Styling"]
        answer: |
          Yes! SkayaJS components are fully compatible with MUI, Tailwind, or any CSS-in-JS library.  
          You can pass `className` props or wrap them in your theme providers.

  - title: "Components & Hooks"
    items:
      - question: "Which components are available?"
        tags: ["Components"]
        answer: |
          Current Navigator-based components:
          <ul>
            <li><strong>GeoLocation</strong> – Show or track user location</li>
            <li><strong>BluetoothConnector</strong> – Connect and manage Bluetooth devices</li>
            <li><strong>Clipboard</strong> – Read/write system clipboard</li>
            <li><strong>MediaDevices</strong> – Access camera/microphone streams</li>
          </ul>
      - question: "Do components have corresponding hooks?"
        tags: ["Hooks"]
        answer: |
          Yes. Each component has a hook, e.g.:
          <ul>
            <li>`useGeoLocation`</li>
            <li>`useBluetooth`</li>
            <li>`useClipboard`</li>
            <li>`useMediaDevices`</li>
          </ul>
          Hooks provide fine-grained control if you want to handle logic separately from UI.

  - title: "Theming & Styling"
    items:
      - question: "How do I enable dark/light mode?"
        tags: ["Theming"]
        answer: |
          Wrap your app with `SkayaThemeProvider` and set `initialTheme` to `"dark"` or `"light"`:
          ```
          <SkayaThemeProvider initialTheme="dark">
            <App />
          </SkayaThemeProvider>
          ```
          Components automatically adapt their styles. You can override using `className` or your CSS framework.
      - question: "Can I change the theme dynamically?"
        tags: ["Theming"]
        answer: |
          Yes! Use the `useSkayaTheme` hook:
          ```
          const { theme, setTheme } = useSkayaTheme();
          <button onClick={() => setTheme(theme === "light" ? "dark" : "light")}>Toggle Theme</button>
          ```

  - title: "Development & Testing"
    items:
      - question: "How do I test SkayaJS components?"
        tags: ["Testing"]
        answer: |
          SkayaJS includes test files for each component.  
          Install dev dependencies and run:
          ```
          npm test
          ```
          or
          ```
          yarn test
          ```
          Example: `GeoLocation` test mocks `navigator.geolocation`.

      - question: "Can I develop locally before publishing?"
        tags: ["Development"]
        answer: |
          Yes, use `npm link` or a local example app:
          ```
          cd skayajs
          npm run build
          npm link
          cd ../my-app
          npm link skayajs
          ```
