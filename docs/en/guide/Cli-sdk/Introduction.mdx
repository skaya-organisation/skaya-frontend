## Skaya SDK 🚀

Skaya is an advanced CLI toolkit that supercharges your full-stack web3 project setup with production-ready templates and interactive scaffolding via Artificial Intelligence.

### Installation 📦

```bash
npm install -g skaya
```

> 🚀 **Frontend generation is fully complete and production-ready.**
> 🔧 **Backend module generation is under active development.**
> 🔧 **Blockchain module generation is under active development.**

### 🧩 AI-Powered Code Generation

#### ✅ Frontend Component Generator

- Smart, context-aware auto-generated component generation via AI
- **Language** TypeScript for compatibility (tsx)
- Uses Modern tailwindcss for styling
- Configure props, state, and component behavior dynamically
- Automatically generate:
  - ✅ **Stories** for each component (Storybook-ready)
  - ✅ **Tests** with sample unit test templates (Jest or Vitest)
