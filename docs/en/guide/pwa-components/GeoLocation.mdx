# üìç GeoLocation Component

The `GeoLocation` component wraps `navigator.geolocation` into a themed React component and works seamlessly with the `useGeolocation` hook.  

## Demo

Here is a live, interactive React component embedded right into this documentation page.

## Component Usage

```tsx
import { useGeolocation, GetLocationButton } from "skayajs";

function Page() {
  const { location, loading, getPosition } = useGeolocation();

  return (
    <div>
      <GetLocationButton
        label="Get My Location"
        loadingLabel="Locating..."
        onGetPosition={getPosition}
        isLoading={loading}
      />

      {location.error && <p style={{ color: "red" }}>{location.error}</p>}
      {location.latitude && location.longitude && (
        <p>
          Latitude: {location.latitude.toFixed(5)}, Longitude: {location.longitude.toFixed(5)}
        </p>
      )}
      {!location.error && !location.latitude && !loading && <p>Click the button to fetch location.</p>}
    </div>
  );
}
```

## Props

| Prop           | Type                                           | Default                     | Description                                      |
|----------------|-----------------------------------------------|----------------------------|--------------------------------------------------|
| `label`        | `string`                                      | `"Get Location"`           | Text displayed on the button                     |
| `loadingLabel` | `string`                                      | `"Getting Location..."`    | Text displayed when fetching location           |
| `onGetPosition`| `() => void`                                  | `required`                 | Function from `useGeolocation` to fetch location|
| `isLoading`    | `boolean`                                     | `false`                    | Whether the request is in progress              |
| `className`    | `string`                                      | `""`                        | Optional custom CSS classes                      |

## Hook Usage

If you prefer using the hook directly:

```tsx
import { useEffect } from "react";
import { useGeolocation } from "skayajs";

function Page() {
  const { location, loading, getPosition } = useGeolocation();

  useEffect(() => {
    getPosition(); // fetch position on mount
  }, [getPosition]);

  if (loading) return <p>Loading...</p>;
  if (location.error) return <p>Error: {location.error}</p>;

  return (
    <p>
      Latitude: {location.latitude?.toFixed(5)}, Longitude: {location.longitude?.toFixed(5)}
    </p>
  );
}
```
