## Basic Usage: Geolocation Example

SkayaJS provides hooks and controlled components for common browser APIs. Hereâ€™s how to use the useGeolocation hook with the new LocationButton component.

```tsx
import React from "react";
import { useGeolocation, LocationButton } from "skayajs";

export default function App() {
  const { location, isLoading, fetchLocation } = useGeolocation();

  const isLocationAvailable = location.latitude !== null && location.longitude !== null;

  return (
    <div style={{ padding: "1rem", fontFamily: "sans-serif" }}>
      <h2>My GPS Tracker</h2>

      <LocationButton
        onClick={fetchLocation}
        isLoading={isLoading}
        style={{ padding: "0.5rem 1rem", fontSize: "1rem" }}
      >
        Get My Location
      </LocationButton>

      {/* Display position if available */}
      {isLocationAvailable && (
        <p>
          Latitude: {location.latitude!.toFixed(5)}, Longitude:{" "}
          {location.longitude!.toFixed(5)}
        </p>
      )}
    </div>
  );
}
```
