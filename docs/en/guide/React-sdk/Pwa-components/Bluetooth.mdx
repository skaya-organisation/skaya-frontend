The **`BluetoothButton`** component is a controlled button designed to trigger the browser's `navigator.bluetooth` API, working seamlessly with the **`useBluetooth`** hook.

## Component Usage (BluetoothButton)

The `BluetoothButton` leverages the standard `children` prop for its content and follows common React naming conventions for handlers and state.
<Tabs>
  <Tab label="Demo"><BluetoothExample /></Tab>

  <Tab label="Props">
```tsx
{

  // Example usage:
  <BluetoothButton
    onClick={fetchBluetoothDevice}
    isLoading={isLoading}
    className="px-4 py-2 bg-blue-500 text-white rounded"
  >
    Connect Bluetooth
  </BluetoothButton>

  interface BluetoothButtonProps {
    children?: React.ReactNode;          // The content displayed on the button when it's NOT connecting. Default: "Connect Bluetooth"
    onClick: () => void;                 // Function to execute on button click, typically fetchBluetoothDevice from the hook. Required.
    isLoading?: boolean;                 // Boolean state indicating if the Bluetooth request is currently in progress. Default: false
    className?: string;                  // Optional custom CSS classes. Default: ""
  }

}

```
</Tab>
  <Tab label="Demo Code">
```tsx
import { BluetoothButton, useBluetooth } from "skayajs";

export const BluetoothDemo = () => {
  const { bluetooth, isLoading, fetchBluetoothDevice, resetBluetooth } =
    useBluetooth();

return (

<div className="flex flex-col items-center rounded-xl space-y-4 text-center">
      <BluetoothButton
        onClick={fetchBluetoothDevice}
        isLoading={isLoading}
        className={`px-6 py-2 rounded-md ${
                  isLoading
                    ? "bg-gray-400"
                    : "bg-green-600 hover:bg-green-700 text-white"
                }`} >
        Connect to Bluetooth Device
      </BluetoothButton>

      {bluetooth.device && <p>Connected to: {bluetooth.device.name}</p>}
      {bluetooth.error && <p style={{ color: "red" }}>{bluetooth.error}</p>}

      <button
        onClick={resetBluetooth}
        className="px-4 py-2 rounded bg-gray-600 text-white hover:bg-gray-700 transition-colors duration-200"
      >
        Reset
      </button>
    </div>
 );
};

export default BluetoothDemo;

```

## Hooks Usage (useBluetooth)

The `useBluetooth` hook gives you programmatic control over Bluetooth connections without needing a button component.

<Tabs>
  <Tab label="Demo"><HooksExample /></Tab>

  <Tab label="Hooks">
```tsx
  {
  // Example usage:
  const { bluetooth, isLoading, fetchBluetoothDevice, resetBluetooth } = useBluetooth();

  // interface
  interface BluetoothState {
    bluetooth: {
      device: BluetoothDevice | null; // the connected Bluetooth device object, or `null` if not connected
      error: string | null;           // any error message if the connection failed
    };
    isLoading: boolean;                // whether the Bluetooth request is in progress
    fetchBluetoothDevice: () => void; // function to trigger a request to connect to a Bluetooth device
    resetBluetooth: () => void;       // function to reset the state (device â†’ null, error â†’ null) and stop loading
  }

}
```
</Tab>
  <Tab label="Demo Code">
```tsx
import React from "react";
import { useBluetooth } from "skayajs";

const HooksExample = () => {
  const { bluetooth, isLoading, fetchBluetoothDevice, resetBluetooth } =
    useBluetooth();

  return (
    <div className="flex flex-col items-center rounded-xl  space-y-4 text-center">
      <button
        onClick={fetchBluetoothDevice}
        disabled={isLoading}
        className={`px-4 py-2 rounded text-white transition-colors duration-200 ${
          isLoading
            ? "bg-gray-400 cursor-not-allowed"
            : "bg-blue-600 hover:bg-blue-700"
        }`}
      >
        {isLoading ? "Connecting Bluetooth..." : "Connect via Hook"}
      </button>

      {bluetooth.device && (
        <p className="text-gray-800">
          Connected to: {bluetooth.device.name || bluetooth.device.id}
        </p>
      )}

      {bluetooth.error && <p className="text-red-600">{bluetooth.error}</p>}

      <button
        onClick={resetBluetooth}
        className="px-4 py-2 rounded bg-gray-600 text-white hover:bg-gray-700 transition-colors duration-200"
      >
        Reset
      </button>
    </div>
  );
};

export default HooksExample;

```
